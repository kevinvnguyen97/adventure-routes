# Adventure Routes
Adventure routes is a web application that provides an easy way for users to plan any adventure they want by plotting waypoints on the map and the Google Maps API will automatically create the best paths between waypoints. Users can also save waypoints and planned routes in their accounts and share them with other users, in addition to adding a few details on the adventure itself, such as a description and purpose of the trip. Users can comment on other people’s routes and share their own experiences in their trip. In a way, this can be viewed as a social media platform.

# Running the App
1. Download the repo.
2. Download and install nodejs: https://nodejs.org/en/download.
3. Go to command prompt / terminal.
4. Navigate to the root of the repo and run ```npm i -g meteor``` and ```npm i```
5. Create a file `settings.json` in the .deploy folder and follow the example file with the given Google API key.
6. Run ```npm start```
